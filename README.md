
# EscapeRoom

## Opis gry

Gra została wykonana na silniku Unreal Engine 4.27. Jest to gra przygodowa/horror, w której gracze rozwiązują zagadki. Do połączenia użytkowników używana jest platforma Steam.

## Fabuła

Zostaliście wysłani aby zbadać stary nawiedzony dom. Musicie szukać poszlak oraz przedmiotów które skierują was na odkrycie jego tajemnic.
